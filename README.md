# 🚀 CIFAR-10 Image Classification using CNN (PyTorch)

## 📌 Project Overview
This project implements a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset.

---

## 🧠 Model Architecture
- 3 Convolutional Layers
- ReLU Activation
- Max Pooling
- Fully Connected Layers

---

## 📊 Dataset
CIFAR-10 contains 10 classes:
- Airplane ✈️
- Car 🚗
- Bird 🐦
- Cat 🐱
- Deer 🦌
- Dog 🐶
- Frog 🐸
- Horse 🐴
- Ship 🚢
- Truck 🚚

---

## ⚙️ Tech Stack
- Python
- PyTorch
- Torchvision
- Matplotlib
- Scikit-learn

---

## 📈 Results
- Training and Validation implemented
- Achieved ~70–80% accuracy
